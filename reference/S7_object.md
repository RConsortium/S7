# Base S7 class

The base class from which all S7 classes eventually inherit from.

## Usage

``` r
S7_object()
```

## Value

The base S7 object.

## Examples

``` r
S7_object
#> <S7_object> class
#> @ parent     : <NULL>
#> @ constructor: function() {...}
#> @ validator  : function(self) {...}
#> @ properties :
```
